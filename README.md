# Json2.jl

JSON parser/builder
- Juila wrapper of following C modules

	- https://github.com/udp/json-parser
	- https://github.com/udp/json-builder

- Done
	- parse()
	- getValue()
	- getindex()
	- build()
	- Auto-Free
	- setindex!()
	- push!()
	- getkeys()
	- getlength()

- Install
	- Pkg.clone(" $url ")
	- Pkg.build("Json2")
